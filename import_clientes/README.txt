IMPORTAÇÃO DE CLIENTES — USO ÚNICO

Pré-requisito: .NET 9 SDK instalado (o mesmo usado pelo Tikaum-Tech)

1. Coloque o arquivo Excel na pasta import_clientes/ com o nome: clientes.xlsx
2. Certifique-se que o Tikaum-Tech NÃO está rodando durante a importação
3. Clique duas vezes em rodar.bat
4. Aguarde o resumo aparecer na tela

ATENÇÃO:
- O script nunca sobrescreve clientes existentes (verifica duplicatas por nome)
- Clientes com dados incompletos são importados com aviso — revise no sistema depois
- Rode apenas uma vez — rodar de novo ignora os já importados (duplicata)
